# Contributing

We welcome contributions!

## Development Setup

1. Clone the repository.
2. Open the project in Unity 6.2 (6000.2) or later.
3. The server code is in `Packages/org.christowm.unity.mcp/Editor`.

## Running Tests

Run the test suite via **Window > General > Test Runner**.
- **EditorMode**: Tests RPC logic and controllers.
- **PlayMode**: Tests runtime interactions.
